What is the freezing point depression?
The freezing point depression is a drop in the temperature at which the substance freezes that is caused when a samller amount of another, non-volatile substance is added.
How to calculate freezing point depression?
To calculate freezing point depression you should use the following formula:
ΔTD = kF bi
where:
ΔTF = freezing point depression
kF = cryoscopic constant
b = molality
i = Van't Hoff factor
Why use freezing point depression?
The freezing point depression is used in order to prevent radiators from freezing in winter because lowering the freezing point allows the street ice to melt at lower temperatures, preventing the accumulation of ice.
